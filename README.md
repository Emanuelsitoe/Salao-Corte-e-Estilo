# Salão de Beleza - Corte e Estilo

## Descrição
Este é um projeto de site para um salão de beleza que oferece serviços de cortes de cabelo, tratamentos estéticos e venda de produtos relacionados. O site é uma página única (single-page) com funcionalidade de agendamento online.

## Funcionalidades
- **Agendamento Online**: Permite aos clientes marcar horários para serviços.
- **Serviços Oferecidos**: Cortes de cabelo, tratamentos capilares e estéticos.
- **Venda de Produtos**: Seção para compra de produtos de beleza.
- **Design Responsivo**: Compatível com dispositivos móveis e desktop.

## Tecnologias Utilizadas
- **HTML**: Estrutura da página.
- **CSS**: Estilização e layout (arquivo `css/index.css`).
- **JavaScript**: Interatividade e funcionalidades (arquivo `js/index.js`).

## Como Executar
1. Clone o repositório:
   ```
   git clone <url-do-repositorio>
   ```
2. Abra o arquivo `index.html` em um navegador web.
